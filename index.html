Coyote

An intelligent command-line tool for
combining and compressing your JavaScript files.

Coyote concatenates your JS files, combining them into a single file with the option of running the output through the Google Closure Compiler before save. Coyote automatically observes your source files for changes and will recompile on the fly for easy development.

Installation

$ gem install coyote

Configuration

$ cd myproject/scripts
$ coyote generate

This will create the configuration file at coyote.yaml and will find any .js files and automatically add them to the input parameter. Open coyote.yaml and modify the input and output configurations to meet your needs. Coyote will combine the input files in the order you define.

Usage

$ coyote

This will start the Coyote "server" -- the directory watcher mechanism. One started, any changes made to your source files will automatically trigger the compiler.

Running Manually

$ coyote build

This will trigger the compiler without the "server" for a one-time build and save.

Input Wildcards

You can wildcard your input parameters to find files.

- **/*.js # recursively find all files with the extension '.js'
- /jquery/plugins/*.js # find all .js files in the directory

This will trigger the compiler without the "server" for a one-time build and save.

Dependency Discovery

To include a dependency, put a require comment in the top of your JS file

If your JavaScript files depend on other files or libraries to run, you can define those dependencies so Coyote will include them in the compiled output before the files that require them.

// require /jquery/jquery.js
// require jquery

When you don't define an explicit path to the file, as in the example above, Coyote will recursively search for a file of that name. "// require jquery" will search the entire directory for the file "jquery.js"

Dependencies are searched for relative to the top level of your directory -- where your coyote.yaml config file is located.

Options

Force compression

$ coyote -c
$ coyote build -c
